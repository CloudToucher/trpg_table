# Boss遭遇战：场景与剧情

> 每章设有一个可选Boss遭遇。Boss战是获取高价值战利品的主要途径。玩家可以选择挑战或绕行，但绕行意味着放弃战利品和经验。Boss遭遇的触发、战斗环境和战利品详见下文。Alpha实验体（终章Boss）的场景见 `scenes/laboratory/星辰实验室.md`。

## BOSS-001：隧道霸主·铁壳蜈蚣

### 章节：第一章（隧道行军）
### 触发位置：SUB-002 与 SUB-003 之间的废弃维修段
### 威胁等级：高危

---

### 剧情背景

地铁隧道在这一段突然变窄。铁轨被某种巨大的力量扭曲、拱起，墙壁上布满了深深的抓痕和凝固的酸蚀痕迹。空气中弥漫着刺鼻的化学气味。地面上散落着被啃食殆尽的丧尸残骸——骨头上的咬痕整齐而巨大，像是被工业剪刀切过。

隧道深处传来有节奏的"咔嗒咔嗒"声，像是无数硬壳敲击铁轨的声音。

### 触发条件

- **必然触发**：沿主隧道前进至维修段时，发现隧道被巨大的茧状物堵住，只能通过维修段绕行
- **提前预警**：PER检定DC标准——成功者注意到墙壁上的酸蚀痕迹和地面的巨型节肢印记，判断出前方有大型变异生物
- **可选绕行**：电子检定DC极难打开一扇锁死的侧门，通过通风管道绕过此区域（放弃Boss战）

### 战前环境

```
        ┌──────────────────────┐
        │    维修段（宽8m）      │
   入口→│                      │→出口（被茧堵住）
        │  ■柱  铁壳蜈蚣巢穴  ■柱 │
        │      【蜈蚣】         │
        │  ■柱    ■工具架   ■柱 │
        │                      │
        │    积水区(浅水)        │
        └──────────────────────┘
```

- **维修段**：宽约8米，长约20米，比普通隧道宽敞但仍是封闭空间
- **支撑柱×4**：可作为掩体（HP20，被蜈蚣冲撞可摧毁）
- **工具架**：可搜索获得铁管、扳手等临时武器
- **积水区**：浅水区域，经过时AGI检定DC普通否则溅水发出噪音
- **废弃维修车**：可推动（STR DC困难）撞击蜈蚣造成2d10伤害+击退
- **天花板管道**：AGI检定DC困难可攀爬，蜈蚣无法攻击到（但会喷酸）
- **电气箱**：电子检定DC标准可使其短路，对积水区所有目标造成2d8电击

### 战斗流程

**开场**：
> "你们进入维修段时，那有节奏的声音突然停了。寂静。然后——天花板上的灰尘簌簌落下，一道巨大的暗影从上方的管道缝隙中滑落。四米长的节肢身躯重重砸在铁轨上，铁锈色的甲壳在应急灯下泛着冷光。数十对细长的足爪敲击地面，发出令人牙酸的金属摩擦声。它的头部缓缓转向你们——两排复眼反射出红色的微光。"

**阶段一（HP>60）——试探**：
- 蜈蚣在柱子间穿行，利用体型优势压迫空间
- 每2轮交替使用颚咬和体节冲撞
- DM提示：此时可观察到腹部甲壳较薄（PER DC标准）

**阶段二（HP 30-60）——狂躁**：
- 蜈蚣受伤后发出尖锐的嘶鸣，从身体裂缝中涌出1d4只蜈蚣幼虫
- 开始喷射酸液，优先攻击对它造成最多伤害的角色
- 移动更加快速和不规则
- 幼虫数据：HP8，护甲0，攻击1d4+感染，经验值3

**阶段三（HP<30）——垂死挣扎**：
- 蜈蚣翻转身体暴露腹部（护甲降为0），进入疯狂攻击
- 每轮2次攻击行动，但防御大开
- 如果此时利用环境（推维修车/电气箱）可快速结束战斗

### 战利品

**固定掉落**：
- 蜈蚣甲壳碎片×3（制作材料：可制作护甲6防具或护甲3全身甲）
- 经验值60（每名参战角色）

**随机掉落**（投Boss战利品表1d12，见物品与掉落表第六节）

**环境击杀奖励**（利用维修车/电气箱/管道等环境击杀）：
- 额外投一次Boss战利品表
- 蜈蚣完整甲壳×1（★★★传说制作材料：可制作护甲8重甲，AGI-2）

**巢穴搜索**（战后可搜索蜈蚣巢穴）：
- 求生检定DC标准
- 成功：发现被蜈蚣收集的"闪亮物品"——手表、金属零件、以及1件稀有物品（投稀有物品表）
- 大成功：额外发现一个被酸液半溶解的军用箱，内有步枪弹药×15、军用急救包×1

## BOSS-002：城市猎手·蔓生巨兽

### 章节：第二章（地面穿越）
### 触发位置：中央公园废墟
### 威胁等级：极危险

---

### 剧情背景

穿过商业区后，你们来到了中央公园。曾经的绿地现在是一片诡异的景象——所有植物都疯长了，树木扭曲成不自然的形状，藤蔓像蛇一样覆盖了每一寸地面。空气湿润而温暖，与周围废墟的冰冷形成强烈对比。

公园中心的那棵百年古榕树……已经变成了别的什么东西。

### 触发条件

- **必然触发**：穿越公园是到达西区的最短路径。进入公园后，藤蔓会在身后封锁退路
- **提前预警**：
  - PER检定DC困难——注意到公园边缘的丧尸尸体被藤蔓缠绕吸收
  - 知识（自然/生物）检定DC标准——判断出这些植物受到了病毒变异的影响
- **可选绕行**：从公园外围的商业街绕行（额外消耗4小时+2次随机遭遇表），不触发Boss战
- **NPC线索**：如果之前在超市据点与幸存者交谈，可能得知"公园里有什么东西在长"的情报

### 战前环境

```
        ┌────────────────────────────┐
        │        中央公园             │
        │   🌳  🌳    🌳  🌳         │
        │      藤蔓密集区             │
   入口→│  🌳 ┌──────────┐ 🌳       │→出口
        │     │ 蔓生巨兽  │           │
        │     │ （古榕树） │           │
        │  🌳 └──────────┘ 🌳       │
        │      藤蔓密集区             │
        │   🌳  喷泉(干涸)  🌳       │
        │         花坛(可燃物)         │
        └────────────────────────────┘
```

- **开阔场地**：公园中心约30m×30m的空间，但地面布满藤蔓
- **藤蔓密集区**：困难地形，移动速度减半，每轮开始AGI DC12否则被绊倒
- **周围树木**：可攀爬（AGI DC标准），在树上不受地面菌丝影响但仍在藤蔓攻击范围内
- **干涸喷泉**：石质掩体，护甲8，可站在其中获得半掩护
- **花坛残骸**：枯叶和干草，可点燃——火焰沿藤蔓蔓延，对巨兽持续造成1d6火焰/轮
- **园丁小屋**（地图边缘）：内有园艺工具（剪刀、铁锹）和汽油桶（可制作燃烧瓶或直接泼洒点燃）
- **寄生哨兵**：公园外围有2d4只被菌丝寄生的行尸巡逻（可尝试潜行绕过或先清除）

### 战斗流程

**开场**：
> "古榕树——不，那已经不是树了。当你们走近到20米时，'它'醒了过来。树干裂开，露出内部脉动的暗绿色组织，像是一颗巨大的心脏。数十条粗壮的藤蔓从地面升起，缓慢而有目的地朝你们伸来。空气中弥漫开一股甜腻的孢子气息。地面的菌丝网络开始发出微弱的荧光——它知道你们在哪里。"

**阶段一（HP>80）——领域控制**：
- 巨兽固定不动，用藤蔓远程攻击（8m距离）
- 每轮释放地面菌丝扩大困难地形范围
- 优先束缚最近的目标
- DM提示：玩家可用火焰清除脚下的藤蔓（求生DC普通），暂时创造安全区域

**阶段二（HP 40-80）——分裂**：
- 巨兽发出低沉的振动，2个藤蔓分体从地面钻出
- 分体数据：HP25，攻击1d6藤蔓鞭（5m），无护甲，火焰伤害×2
- 核心开始缓慢移动（2m/轮），试图远离近战角色
- 如果有角色被束缚超过2轮，巨兽会尝试吞噬

**阶段三（HP<40）——核心暴露**：
- 外层藤蔓枯萎剥落，暴露出发光的绿色核心
- 核心护甲降为2，但每轮自动释放孢子爆发（全场8m）
- 所有人每轮需要CON DC14否则中毒
- 集中火力攻击核心可快速结束战斗（核心伤害×2）

### 战利品

**固定掉落**：
- 蔓生核心（异能催化物：触发异能觉醒或升级事件）
- 藤蔓纤维×5（制作材料：可制作绳索、鞭类武器、柔性防具）
- 经验值75（每名参战角色）

**随机掉落**（投Boss战利品表1d12）

**环境击杀奖励**（利用火焰大规模焚烧击杀）：
- 额外投一次Boss战利品表
- 净化孢子×2（消耗品：清除感染值2点，或对菌丝类敌人造成3d8真实伤害）

**公园搜索**（战后藤蔓枯萎，可搜索公园）：
- 求生检定DC标准
- 成功：在被藤蔓缠绕的尸体上找到物资（投住宅区掉落表×2）
- 大成功：在古榕树根部发现一个被保护的地下空间，内有某个研究员的背包——含实验室相关笔记（S10碎片）和1剂实验性药物（投实验性药物表）

## BOSS-003：西区守卫·钢筋傀儡师

### 章节：第三章（西区废墟）
### 触发位置：WEST-001 工业区·废弃钢铁厂
### 威胁等级：极危险

---

### 剧情背景

工业区的废弃钢铁厂本该是空无一人的——但你们在接近时注意到了不对劲的地方。厂区外围有排列整齐的丧尸"巡逻"着，它们的动作僵硬而有规律，完全不像正常丧尸的随机游荡。更诡异的是，有些丧尸身上嵌着钢筋和金属碎片，像是被改造过。

有什么东西在控制它们。

### 触发条件

- **侦察触发**：在工业区探索时，PER检定DC标准发现异常的丧尸行为模式
- **NPC线索**：猎人（WEST-002）如果友好，会警告"工厂里有个不一般的东西，它会操控尸体"
- **强制触发**：如果从工厂方向前往实验室，必经此路
- **可选绕行**：通过地下管道网络绕行（AGI检定DC困难×3次，狭窄空间，失败则卡住需STR DC标准脱出），不触发Boss战

### 战前准备（重要）

> 钢筋傀儡师是最有"智慧"的Boss。直接冲入工厂将面对本体+全部傀儡的围攻，极其危险。建议DM引导玩家进行战前侦察和准备。

**可获取的战前情报**：
- 观察工厂1小时（PER DC标准）：了解傀儡巡逻路线，发现约6-8只傀儡
- 猎人的情报：本体在工厂二楼的控制室中，很少亲自出面
- 知识（机械）检定DC困难：推断菌丝线是操控关键，切断可瘫痪傀儡

**可执行的战前行动**：
- 设置陷阱（见制作系统）
- 清除外围巡逻傀儡（潜行击杀，不会惊动本体，除非触发爆破傀儡的爆炸）
- 切断工厂电力（电子DC困难）——工厂陷入黑暗，傀儡师感知范围减半但玩家也需要光源
- 从侧门/窗户潜入，绕过大厅直奔二楼

### 战斗环境

```
        ┌─────────────────────────────┐
        │  钢铁厂二楼（控制室+走廊）     │
        │  ┌────────┐                  │
        │  │傀儡师本体│  走廊            │
        │  │控制室   │←楼梯             │
        │  └────────┘                  │
        ├─────────────────────────────┤
        │  钢铁厂一楼（厂房大厅）        │
   入口→│                              │
        │  ■废铁堆  ■传送带  ■废铁堆    │
        │      钢筋傀儡×2               │
        │  ■熔炉(熄灭)  行尸傀儡×4      │
        │      爆破傀儡×2（隐藏）        │
        │  ■叉车    ■钢板堆             │
        └─────────────────────────────┘
```

- **厂房大厅**：约25m×15m，高度8m（二层结构）
- **废铁堆**：半掩护，可搜索获得金属制作材料
- **传送带**：可启动（电子DC标准），启动后每轮自动将传送带上的物体移向熔炉方向
- **熔炉**：已熄灭，但可重新点燃（求生DC困难+燃料）——点燃后范围3m内每轮2d6火焰伤害
- **叉车**：可驾驶（机械DC标准），冲撞造成3d8伤害+击飞，但噪音巨大
- **钢板堆**：可推倒（STR DC困难），范围3m内2d8砸击伤害
- **楼梯**：通往二楼控制室，宽度仅容2人并排——傀儡师会在此设置钢筋傀儡堵路
- **控制室窗户**：可从外部攀爬进入（AGI DC极难），或用爆炸物炸开墙壁

### 战斗流程

**开场（取决于玩家的进入方式）**：

*正面进入*：
> "你们推开工厂的铁门时，锈蚀的铰链发出刺耳的尖叫。大厅中央，四具'丧尸'同时转头看向你们——动作整齐划一，像是被同一根线牵动。黑暗中，更多的红点亮起。二楼走廊的栏杆后，一个瘦长的身影缓缓站起。你听到了一个沙哑的、像是被石子堵住的声音——它在……笑？"

*潜行进入*：
> "你们从侧窗翻入工厂。借着微弱的月光，你们看到大厅中的傀儡正机械地重复着巡逻路线。它们还没有发现你们——但每只傀儡的背部都连接着一根几乎看不见的暗绿色细丝，向上延伸至二楼。那根丝的终点……就是你们的目标。"

**阶段一（HP>70）——傀儡攻势**：
- 傀儡师躲在控制室中，完全通过傀儡作战
- 每轮使用"操控重组"维持傀儡数量，或"全体指挥"强化攻击
- 钢筋傀儡守住楼梯口，行尸傀儡包围玩家
- 爆破傀儡在玩家聚集时冲入自爆
- **关键**：需要找到方法突破傀儡防线接近本体，或切断菌丝线瘫痪傀儡

**阶段二（HP 35-70）——恐慌**：
- 本体受到威胁后，傀儡数量上限+2
- 开始使用精神尖啸（范围10m SP伤害）
- 本体会通过控制室后门尝试转移到厂房另一侧
- 菌丝线变得更粗更明显（PER DC标准即可发现）

**阶段三（HP<35）——融合**：
- 傀儡师发出绝望的嘶吼，与最大的钢筋傀儡融合
- 变为巨型形态：HP+40（在剩余HP基础上），护甲6，近战2d10
- 不再能操控其他傀儡（剩余傀儡失去控制，原地呆滞1轮后恢复为普通丧尸行为）
- 融合体笨重但极具破坏力，会追击对它伤害最高的角色

### 战利品

**固定掉落**：
- 傀儡丝线×1（消耗品：可操控至多2只丧尸为己方战斗1小时）
- 傀儡师的菌丝核心（制作材料：可制作菌丝类装备，DM裁决）
- 工程师的工具箱（★★精良工具箱：所有制作检定+15）
- 经验值80（每名参战角色）

**随机掉落**（投Boss战利品表1d12）

**战术击杀奖励**（利用战前准备/潜行/环境获得显著优势击杀）：
- 额外投一次Boss战利品表
- 傀儡操控手套（★★★传说：每天可操控1只丧尸为己方战斗，持续至丧尸被摧毁，不消耗）

**工厂搜索**（战后可搜索整个工厂）：
- 机械检定DC标准
- 成功：工厂仓库中有大量制作材料（金属片×10、铁丝×5、螺丝螺母×10、工具零件×5）
- 控制室搜索：傀儡师生前的笔记本——记录了它作为工程师时对实验室的了解（实验室侧门位置+部分安保系统弱点情报）
- 大成功：在熔炉旁发现一把被傀儡师"收藏"的武器——切割锯（★★精良近战武器：2d6+2切割伤害，暴击时肢体切断）

## Boss遭遇通用规则

### 逃跑规则

Boss战中玩家可以选择撤退：
- 需要全队同时脱离Boss的感知范围（通常为20-30m）
- 撤退时Boss会进行借机攻击
- 成功撤退后Boss不会追出自己的领地
- 撤退后可以重新准备再次挑战（Boss恢复至满HP）

### 难度调节建议（DM参考）

| 队伍情况 | 调节方式 |
|----------|----------|
| 3人以下小队 | Boss HP×0.7，傀儡/分裂体数量-1 |
| 装备精良/异能强力 | Boss阶段二提前触发，增加小怪数量 |
| 队伍状态差（低HP/缺物资） | 提供更多环境互动选项，增加战前准备机会 |
| 新手玩家 | DM主动提示弱点和环境利用方式 |

### 死亡与失败

- Boss战中角色死亡遵循核心规则的死亡规则
- 全队阵亡（TPK）时，DM可选择：
  - Boss将玩家拖回巢穴（苏醒后失去部分装备，但有逃跑机会）
  - NPC救援（如猎人路过、军方残部经过）
  - 时间倒流至战前，允许更多准备
