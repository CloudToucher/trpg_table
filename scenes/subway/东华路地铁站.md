# 场景：东华路地铁站（起始点）

## 场景ID：SUB-001
## 章节：序章
## 威胁等级：安全（站内）/ 危险（站外）

---

## 场景概览

东华路站是一座标准的地下两层地铁站。灾变后被约30名幸存者占据并改造成临时避难所。站台层被用作生活区，站厅层的出入口用翻倒的售货机和金属栅栏封堵。应急灯提供昏暗的照明，空气循环系统仍在运转但越来越差。

---

## 区域描述

### A. 站厅层（入口层）

**环境描述**：
"你站在地铁站的站厅层。头顶的应急灯发出暗黄色的光，在地面的积水中映出扭曲的倒影。北面是用售货机和铁栅栏堆成的路障——那是通往地面的出口，被封得严严实实。几个值班的幸存者握着铁管和球棒，警惕地看向路障外面偶尔传来的呻吟声。南面是通往站台层的自动扶梯，早已停运，变成了普通的楼梯。"

**可互动对象**：
- 路障：可以检查（搜索检定），了解防御状况
- 售货机：已被搜刮，但角落可能还有东西（搜索DC困难）
- 值班人员：可以对话获取外面的情况
- 站厅地图：墙上贴着地铁线路图，可以研究路线
- 储物柜：几个行李储物柜，部分上锁（锁技检定DC标准）

### B. 站台层（生活区）

**环境描述**：
"下到站台层，景象更加'生活化'。轨道两侧的站台被铺满了睡袋、毯子和从附近超市搜来的杂物。左侧站台是'生活区'——人们围坐在用铁桶做的简易火炉旁，低声交谈。右侧站台是'物资区'——王姐管理着为数不多的集体物资。站台尽头，通向隧道的黑暗入口像张开的巨口，被一道简易铁门阻隔着。"

**可互动对象**：
- 老陈的角落：避难所管理者的位置（NPC交互）
- 小林的位置：受伤侦察兵躺在角落（NPC交互）
- 王姐的物资区：可以交易的区域（NPC交互）
- 简易火炉：周围有其他幸存者可以交谈
- 隧道入口：离开避难所的通道（需要决定出发）
- 公告板：贴着一些幸存者留言和信息

### C. 维修室（隐藏区域）

**发现条件**：搜索检定DC困难(-10)

**环境描述**：
"在站台层的角落，一扇不起眼的门后是地铁维修室。里面堆满了工具、备用零件和维修设备。角落里有一个工具柜，上面落满了灰尘。"

**可获得物资**：
- 撬棍×1
- 扳手×1
- 手电筒（大型）×1
- 电池×1d4
- 工作手套（格挡时减少1点伤害）
- 安全帽（头部防护+1）

## 事件触发

### 触发条件表

| 条件 | 事件 |
|------|------|
| 玩家与老陈交谈 | 获取地铁线路信息和地面情报 |
| 玩家治疗小林 | 获得详细隧道地图和好感 |
| 玩家查看公告板 | 发现支线任务S01「失踪的孩子」线索 |
| 玩家搜索维修室 | 获得工具类物资 |
| 玩家准备出发时 | 老陈给予临别物资（2个罐头、2瓶水） |
| 玩家在站内休息 | 安全恢复100%，不会被袭击 |
| 游戏内时间超过12小时未出发 | 物资开始紧张，NPC催促 |

## 离开方式

1. **从隧道出发**（推荐）：打开站台尽头的铁门进入隧道
2. **从地面出发**：搬开站厅层路障从地面走（更危险，但可能有更多物资）

玩家选择后进入第一章。
