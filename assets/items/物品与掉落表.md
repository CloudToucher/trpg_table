# 物品与掉落表

## 一、搜索物资系统

### 搜索流程

1. 玩家声明搜索特定区域
2. DM根据区域类型确定掉落表
3. 玩家进行搜索检定（PER + 搜索技能）
4. 根据检定结果和掉落表给出物资

### 搜索检定结果

| 检定结果 | 效果 |
|----------|------|
| 大成功(01-05) | 掉落表投两次，取好的，额外发现稀有物品 |
| 成功 | 掉落表正常投骰 |
| 临界成功 | 掉落表投骰，物品数量-1（最低1） |
| 失败 | 找到1件常见物品或什么也没找到 |
| 大失败(96-00) | 触发陷阱/丧尸/噪音 |

## 五、制作系统（动态框架）

> **核心原则：制作是开放式的。** 玩家可以尝试用手头的任何材料制作任何合理的物品。DM根据材料、环境、角色技能即时判定是否可行，并设定配方、检定难度和成品效果。以下规则和示例仅作为DM的参考锚点。

### 5.1 动态制作流程

```
1. 玩家声明想要制作的物品，描述思路和使用的材料
2. DM判断方案是否合理（考虑现实逻辑和游戏平衡）
3. DM设定：
   - 所需材料（玩家提出的+DM认为必要的）
   - 检定技能和难度（参考下方难度锚点表）
   - 成品效果（参考下方效果锚点表）
   - 制作时间（简单物品：5分钟 / 复杂物品：30分钟~1小时）
4. 玩家进行检定
5. 根据结果生成成品
```

### 5.2 DM判定锚点

#### 难度锚点（根据成品复杂度和创意合理性选取）

| 复杂度 | 难度 | 典型场景 |
|--------|------|----------|
| 简单拼合 | 普通(+10) | 把刀绑在棍子上、用布条做火把 |
| 基础制作 | 标准(±0) | 燃烧瓶、绊线陷阱、简单包扎工具 |
| 需要技巧 | 困难(-10) | 简易防具、武器改造、药品调配 |
| 高度复杂 | 极难(-20) | 消音器、弩、电子设备改装 |
| 突破常识 | 噩梦(-30) | 简易发电机、化学武器、伪装服 |

#### 效果锚点（成品数值的参考范围）

| 类别 | 弱效 | 中效 | 强效 | 超强（极难以上才可能） |
|------|------|------|------|----------------------|
| 武器伤害 | 1d4 | 1d6~1d8 | 1d10~2d6 | 2d8+ |
| 护甲值 | 1~2 | 3~4 | 5~6 | 7+ |
| 治疗量 | 1d4 HP | 1d8~2d6 HP | 2d8~3d6 HP | 全恢复 |
| 范围效果 | 2m | 3~5m | 5~8m | 10m+ |
| 持续时间 | 1轮 | 1d3轮 | 3~5轮 | 10轮+ |
| 检定加值 | +5 | +10 | +15 | +20 |

#### 检定结果对成品的影响

| 检定结果 | 成品效果 |
|----------|----------|
| 大成功(01-05) | 成品效果上浮一档（如中效→强效），额外附加一个有利特性 |
| 成功 | 按DM预设效果正常生成 |
| 临界成功 | 成品可用但效果下浮一档，或有小缺陷（耐久减半等） |
| 失败 | 制作失败，50%几率损失1份主要材料 |
| 大失败(96-00) | 事故！可能造成：爆炸(1d6自伤)/噪音(引来丧尸)/材料全毁 |

### 5.3 特殊制作规则

- **环境要求**：制作需要相对安全的环境。战斗中无法制作（但可以用次要行动做极简单的操作，如点燃已准备好的燃烧瓶）
- **工具加成**：拥有工具箱/对应工具时，检定+10
- **协作制作**：队友可以协助（被协助者检定+5，每个额外协助者再+3，上限+11）
- **即兴创作**：玩家提出特别有创意但合理的方案，DM可给予+5"创意加值"
- **拆解回收**：可以拆解现有物品获取材料（机械检定DC标准，失败毁坏物品）

## 六、Boss战利品系统

> 每章设有一个可选的高难度Boss遭遇战。击败Boss后必定掉落高价值战利品，是获得传说/神器级物品的主要途径。Boss遭遇战的详细场景和剧情见 `scenes/special/Boss遭遇战.md`。

### 战利品规则

- Boss击杀后掉落 **固定战利品 + 随机战利品**
- 固定战利品为该Boss的标志性物品，必定掉落
- 随机战利品投 **Boss战利品表**（见下方）
- 如果Boss被特殊方式击杀（利用弱点/环境击杀），额外投一次随机战利品

### 各章Boss及固定战利品

| 第二章 | 城市猎手·蔓生巨兽 | 蔓生核心（异能催化物：触发异能觉醒或升级事件） | ★★★传说 |
