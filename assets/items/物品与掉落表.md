# 物品与掉落表

## 一、搜索物资系统

### 搜索流程

1. 玩家声明搜索特定区域
2. DM根据区域类型确定掉落表
3. 玩家进行搜索检定（PER + 搜索技能）
4. 根据检定结果和掉落表给出物资

### 搜索检定结果

| 检定结果 | 效果 |
|----------|------|
| 大成功(01-05) | 掉落表投两次，取好的，额外发现稀有物品 |
| 成功 | 掉落表正常投骰 |
| 临界成功 | 掉落表投骰，物品数量-1（最低1） |
| 失败 | 找到1件常见物品或什么也没找到 |
| 大失败(96-00) | 触发陷阱/丧尸/噪音 |
