# 丧尸图鉴

## 丧尸分类

### 一级：普通丧尸

#### 【行尸】- 最常见
| 属性 | 值 |
|------|-----|
| HP | 15 |
| STR | 4 |
| AGI | 2 |
| CON | 5 |
| PER | 2（听觉4） |
| 攻击 | 爪击 1d6，咬击 1d8+感染 |
| 护甲 | 0 |
| 先攻 | 2+2+1d10 |
| 经验值 | 5 |

**特性：**
- 缓慢但不知疲倦
- 主要依靠听觉和嗅觉
- 单只不危险，成群致命
- 受到头部伤害时伤害×2

**弱点：** 头部（命中需额外-10，但伤害×2）
**行为模式：** 缓慢游荡，听到声音后朝声源移动

#### 【奔跑者】- 新鲜感染
| 属性 | 值 |
|------|-----|
| HP | 12 |
| STR | 5 |
| AGI | 6 |
| CON | 4 |
| PER | 4 |
| 攻击 | 扑击 1d8+感染，冲撞 1d6+击倒 |
| 护甲 | 0 |
| 先攻 | 6+4+1d10 |
| 经验值 | 10 |

**特性：**
- 感染不到48小时的个体，保留了运动能力
- 速度快，会奔跑追击
- 比行尸脆弱但更难命中
- 通常单独或2-3只出现

**弱点：** 头部，体力有限（追击超过5轮后AGI-2）
**行为模式：** 发现猎物后全速冲刺

### Boss级：各章节Boss

#### 【铁壳蜈蚣】（第一章Boss）
| 属性 | 值 |
|------|-----|
| HP | 90 |
| STR | 8 |
| AGI | 5 |
| CON | 9 |
| PER | 4（震动感知10m） |
| 护甲 | 6（腹部3） |
| 先攻 | 5+4+1d10 |
| 经验值 | 60 |

**攻击模式（每轮选择一种）：**
| 攻击 | 伤害 | 特性 |
|------|------|------|
| 颚咬 | 2d8+感染 | 命中后自动进入撕咬（下轮自动1d8，STR对抗脱出） |
| 体节冲撞 | 1d10+击退3m | 直线5m范围，可撞击多个目标 |
| 酸液喷射 | 2d6酸蚀 | 锥形3m，护甲永久-1（被命中的装备） |
| 蜷缩防御 | — | 护甲变为10，但无法攻击，持续1轮 |

**阶段机制：**
- **阶段一（HP>60）**：正常攻击，每2轮在颚咬和冲撞间交替
- **阶段二（HP 30-60）**：开始使用酸液喷射，移动速度+2m，召唤1d4只小型蜈蚣幼虫（HP8，攻击1d4+感染）
- **阶段三（HP<30）**：狂暴——每轮2次攻击行动，但腹部弱点完全暴露（护甲0）

**弱点：** 腹部（护甲减半，需要侧翼/翻转攻击，命中-10）；火焰（火焰伤害×1.5）
**行为模式：** 在隧道中高速移动，利用狭窄空间限制玩家走位。优先攻击最近的目标。

**背景：** 地铁隧道深处的巨型变异节肢生物，由数十只被感染的蜈蚣融合而成，体长约4米。甲壳坚硬如铁，以隧道中的丧尸和老鼠为食。它将一段隧道视为自己的巢穴，任何进入者都会被视为猎物。

#### 【蔓生巨兽】（第二章Boss）
| 属性 | 值 |
|------|-----|
| HP | 120 |
| STR | 7 |
| AGI | 6 |
| CON | 8 |
| PER | 8（菌丝感知20m） |
| INT | 3（本能策略） |
| 护甲 | 4（核心区域8） |
| 先攻 | 6+8+1d10 |
| 经验值 | 75 |

**攻击模式（每轮选择一种）：**
| 攻击 | 伤害 | 特性 |
|------|------|------|
| 藤蔓鞭挞 | 2d8 | 攻击距离8m，可同时攻击2个目标 |
| 藤蔓缠绕 | 1d6/轮（持续） | 命中后束缚，STR DC16或AGI DC14脱出 |
| 孢子爆发 | 1d8毒素+感染 | 范围8m，CON DC14否则中毒（每轮HP-3，持续3轮） |
| 吞噬 | 3d8 | 对已束缚目标，将其拖入体内消化 |
| 地面菌丝 | — | 指定一片5m×5m区域变为困难地形，经过者AGI DC12否则减速 |

**阶段机制：**
- **阶段一（HP>80）**：用藤蔓远程攻击，保持距离，用地面菌丝控场
- **阶段二（HP 40-80）**：分裂出2个藤蔓分体（HP25，攻击1d6藤蔓鞭），核心开始移动
- **阶段三（HP<40）**：核心暴露发光——全力孢子爆发（每轮自动释放），但护甲降为2

**弱点：** 核心（隐藏在藤蔓团中心，命中-15，但伤害×2）；火焰（火焰伤害×2，且烧毁藤蔓使其无法再生）
**行为模式：** 控制型——优先束缚落单目标，用地形限制玩家移动，逐个吞噬。

**背景：** 城市中心公园中一棵被病毒感染的古树变异而成。菌丝网络蔓延了整个公园，以吸收有机体为食。它的核心深藏在扭曲的藤蔓之中，像一颗跳动的绿色心脏。周围的丧尸会被菌丝寄生，成为它的"哨兵"。

#### 【钢筋傀儡师】（第三章Boss）
| 属性 | 值 |
|------|-----|
| HP | 100（本体） |
| STR | 6 |
| AGI | 4 |
| CON | 7 |
| PER | 10（通过傀儡感知） |
| INT | 8（高度智能） |
| 护甲 | 3（本体）/ 傀儡护甲见下 |
| 先攻 | 4+10+1d10 |
| 经验值 | 80 |

**特殊机制——傀儡操控：**
傀儡师本体脆弱，通过菌丝操控周围的丧尸尸体和工业废料组成"傀儡"作战。

| 傀儡类型 | HP | 护甲 | 攻击 | 数量上限 |
|----------|-----|------|------|----------|
| 行尸傀儡 | 20 | 0 | 1d6爪击 | 4只 |
| 钢筋傀儡 | 35 | 5 | 2d6重击+击退 | 2只 |
| 爆破傀儡 | 10 | 0 | 死亡爆炸3d6(3m) | 2只 |

**本体攻击模式（每轮选择一种）：**
| 攻击 | 伤害 | 特性 |
|------|------|------|
| 菌丝射线 | 2d6 | 射程15m，命中后目标下轮行动-5 |
| 操控重组 | — | 复活1只已死傀儡或将场地中的材料组成新傀儡 |
| 精神尖啸 | 1d8 SP伤害 | 范围10m，WIL DC15否则恐惧1轮 |
| 全体指挥 | — | 所有傀儡本轮攻击+5，伤害+2 |

**阶段机制：**
- **阶段一（HP>70）**：躲在傀儡群后方，通过傀儡作战，每轮操控重组
- **阶段二（HP 35-70）**：傀儡上限+2，开始使用精神尖啸，本体会尝试转移位置
- **阶段三（HP<35）**：绝望——与最大的钢筋傀儡融合，变为巨型形态（HP+40，护甲6，近战2d10），但不再能操控其他傀儡

**弱点：** 本体（如果能绕过傀儡直击本体，本体护甲仅3且无法闪避）；切断菌丝（破坏连接傀儡的菌丝线可使傀儡瘫痪1轮，PER DC困难发现菌丝线）
**行为模式：** 战术型——保持距离，利用傀儡消耗玩家资源，用爆破傀儡打断玩家阵型。是所有Boss中最有"智慧"的。

**背景：** 曾是西区一家工厂的工程师，感染后变异获得了通过菌丝操控物体的能力。它保留了人类的智慧碎片，会设置陷阱和伏击。工厂区域是它的"领地"，废弃的钢筋、机械和丧尸尸体都是它的素材。

### Boss级：实验体

**攻击模式（每轮选择一种）：**
| 攻击 | 伤害 | 特性 |
|------|------|------|
