# 丧尸图鉴

## 丧尸分类

### 一级：普通丧尸

#### 【行尸】- 最常见
| 属性 | 值 |
|------|-----|
| HP | 15 |
| STR | 4 |
| AGI | 2 |
| CON | 5 |
| PER | 2（听觉4） |
| 攻击 | 爪击 1d6，咬击 1d8+感染 |
| 护甲 | 0 |
| 先攻 | 2+2+1d10 |
| 经验值 | 5 |

**特性：**
- 缓慢但不知疲倦
- 主要依靠听觉和嗅觉
- 单只不危险，成群致命
- 受到头部伤害时伤害×2

**弱点：** 头部（命中需额外-10，但伤害×2）
**行为模式：** 缓慢游荡，听到声音后朝声源移动

---

#### 【奔跑者】- 新鲜感染
| 属性 | 值 |
|------|-----|
| HP | 12 |
| STR | 5 |
| AGI | 6 |
| CON | 4 |
| PER | 4 |
| 攻击 | 扑击 1d8+感染，冲撞 1d6+击倒 |
| 护甲 | 0 |
| 先攻 | 6+4+1d10 |
| 经验值 | 10 |

**特性：**
- 感染不到48小时的个体，保留了运动能力
- 速度快，会奔跑追击
- 比行尸脆弱但更难命中
- 通常单独或2-3只出现

**弱点：** 头部，体力有限（追击超过5轮后AGI-2）
**行为模式：** 发现猎物后全速冲刺

---

### 二级：变异丧尸

#### 【膨胀者】
| 属性 | 值 |
|------|-----|
| HP | 35 |
| STR | 6 |
| AGI | 1 |
| CON | 8 |
| PER | 2 |
| 攻击 | 重击 1d10+击退，死亡爆炸 2d8范围(3m)+感染 |
| 护甲 | 2 |
| 先攻 | 1+2+1d10 |
| 经验值 | 15 |

**特性：**
- 体型膨大，充满腐败气体
- 极慢但非常耐打
- 死亡时爆炸，范围伤害+感染检定
- 远距离击杀为佳

**弱点：** 远程攻击（避免近距离击杀触发爆炸）
**行为模式：** 缓慢接近，试图与目标近身

| 属性 | 值 |
|------|-----|
| HP | 18 |
| STR | 5 |
| AGI | 7 |
| CON | 4 |
| PER | 6 |
| 攻击 | 偷袭 2d8+感染（伏击时伤害+1d6），爪击 1d8 |

**弱点：** 被发现后战斗力大降
**行为模式：** 潜伏等待落单目标，偷袭后视情况撤退或继续

### 三级：高级变异丧尸

#### 【坦克】
| 属性 | 值 |
|------|-----|
| HP | 80 |
| STR | 10 |
| AGI | 3 |
| CON | 10 |
| PER | 3 |
| 攻击 | 重拳 2d10+击飞(5m)，践踏 1d10(范围)，投掷物体 2d8(15m) |
| 护甲 | 5 |
| 先攻 | 3+3+1d10 |
| 经验值 | 50 |

**特性：**
- 巨型变异体，身高约2.5米
- 极高的HP和护甲
- 可以破坏掩体和障碍物
- 可以投掷汽车等大型物体

**弱点：** 背部（护甲减半），速度慢可以风筝
**行为模式：** 直线冲锋，破坏一切障碍

### Boss级：实验体

#### 【Alpha实验体】（终章Boss）
| 属性 | 值 |
|------|-----|
| HP | 200 |
| STR | 12 |
| AGI | 8 |
| CON | 12 |
| PER | 8 |
| INT | 6（保留部分智慧） |
| 护甲 | 8 |
| 先攻 | 8+8+1d10 |
| 经验值 | 100 |

**攻击模式（每轮选择一种）：**
| 攻击 | 伤害 | 特性 |
|------|------|------|
| 利爪连击 | 2d10+4 | 攻击两次 |
| 尾扫 | 1d10+击倒 | 范围攻击(3m) |
| 吞噬 | 3d10 | 需先命中抓取，被抓者需STR对抗脱出 |
| 再生 | 恢复20HP | 每3轮使用一次 |
| 病毒波动 | 2d8感染伤害 | 5m范围，需CON检定否则感染值+2 |

**阶段机制：**
- **阶段一（HP>150）**：正常攻击模式
- **阶段二（HP 80-150）**：速度增加（AGI+3），每轮2次主要行动
- **阶段三（HP<80）**：进入狂暴，所有伤害+5，但护甲-3

**弱点：** 头部后方的暴露核心（需要特定位置攻击，命中-20，但伤害×3且无视护甲）
