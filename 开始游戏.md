# 《末日地铁：无限》- 游戏启动

## 欢迎

这是一个通过AI本地文件读写驱动的TRPG游戏。你将扮演一名丧尸末日中的幸存者，从城市东侧的地铁站出发，穿越危机四伏的城市，到达西侧的实验室寻找血清。

---

## 如何开始

### 新游戏

对AI说：**"开始新游戏"**

AI(DM)将引导你完成以下流程：
1. **角色创建** → 选择职业、分配属性、选择异能
2. **角色确认** → 确认角色卡并保存
3. **序章开始** → 进入东华路地铁站的开场场景

### 继续游戏

对AI说：**"继续游戏"** 或 **"读取存档"**

AI(DM)将：
1. 先执行恢复命令：`python saves/save_manager.py restore -c [campaign_id] [--snapshot 快照ID]`
2. 查找 `saves/` 目录下的最新存档
3. 恢复游戏状态
4. 简要回顾上次进展
5. 从上次的位置继续

## 游戏操作指南

### 基本交互

你可以用自然语言描述你想做的任何事情。例如：

**探索类：**
- "我四处看看"
- "搜索这个房间"
- "检查那个柜子"
- "我打开那扇门"

**移动类：**
- "我往北走"
- "我进入隧道"
- "我爬上楼梯"
- "我回到之前的房间"

**战斗类：**
- "我用撬棍攻击最近的丧尸"
- "我开枪射击"
- "我闪避"
- "我使用[异能名]"

**社交类：**
- "我和老陈说话"
- "我问他关于实验室的事"

- "我使用急救包"

- "查看状态" → 显示当前角色状态

## 游戏提示

### 生存建议
1. **保持安静**：枪械会吸引丧尸，优先使用近战或安静武器
2. **管理物资**：食物和水是有限的，搜索每个区域
3. **观察环境**：经常使用搜索/感知，可能发现隐藏的物资或危险
4. **谨慎前进**：不要贸然冲入未知区域
5. **注意感染**：被丧尸咬伤后及时使用抗生素
6. **善用异能**：但注意SP消耗，不要在非关键时刻浪费

### 战斗建议
1. **避免群战**：5只以上的丧尸群应考虑绕行
2. **瞄准头部**：虽然难度更高，但伤害翻倍
3. **优先嚎叫者**：它会不断召唤更多丧尸
4. **利用地形**：高处、瓶颈、掩体都是你的朋友
5. **保留SP**：异能是关键时刻的救命手段
