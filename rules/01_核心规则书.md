# 《末日地铁：无限》核心规则书

## 版本：1.0 | 系统代号：METRO-INF

---

## 第一章：数值体系

### 1.1 基础属性（共6项，初始点数池：30点）

每项属性范围 **1-10**（人类常规极限），超过10需要异能或特殊装备加成。

| 属性 | 缩写 | 描述 | 影响领域 |
|------|------|------|----------|
| **力量** | STR | 物理力量、近战伤害、负重 | 近战攻击、破坏、搬运 |
| **敏捷** | AGI | 速度、反应、闪避 | 闪避、先攻、远程命中 |
| **体质** | CON | 耐久、生命、抗性 | 生命值、毒/病抗性、耐力 |
| **感知** | PER | 观察、直觉、搜索 | 侦查、陷阱发现、伏击察觉 |
| **智力** | INT | 分析、记忆、技术 | 技术操作、解谜、异能控制 |
| **意志** | WIL | 精神力、抗压、异能潜力 | 精神防御、异能强度、恐惧抗性 |

### 1.2 衍生数值

| 数值 | 计算公式 | 说明 |
|------|----------|------|
| **生命值 HP** | CON × 5 + 10 | 降至0则濒死，-10则死亡 |
| **精神值 SP** | WIL × 4 + INT × 2 | 使用异能和承受恐惧消耗 |
| **耐力值 EP** | CON × 3 + STR × 2 | 奔跑、格挡、连续战斗消耗 |
| **先攻值** | AGI + PER + 1d10 | 每轮战斗开始时投骰决定行动顺序 |
| **近战攻击** | STR×10 + 技能×5 + (AGI+PER)×2 | 敏捷=出手速度,感知=捕捉破绽 |
| **远程攻击** | AGI×10 + 技能×5 + (AGI+PER)×2 | 敏捷=瞄准,感知=预判轨迹 |
| **闪避值** | AGI + PER/2(向下取整) | 被动闪避门槛 |
| **负重上限** | STR × 5 (kg) | 超出则AGI-2，速度减半 |

### 1.3 伤害与治疗

**伤害类型：**
- **物理伤害**：受护甲减免
- **异能伤害**：受精神抗性减免
- **真实伤害**：无法减免（毒素、诅咒等）

**濒死与死亡：**
- HP降至0：濒死状态，每轮投1d10，≥6则稳定，＜6则HP-1
- HP降至-10：角色死亡
- 濒死状态可被治疗/急救拉回（INT检定DC12）

**自然恢复：**
- 休息1小时：恢复 CON 点HP，WIL 点SP，CON+STR 点EP
- 完整睡眠（4小时以上）：恢复 50% 最大HP/SP/EP
- 进食+饮水：额外恢复 10HP，5EP
- 短休（冥想）：15-30分钟静坐冥想，恢复 WIL×2 点SP。每次长休（1小时+）之间最多2次。战斗中/行进中不可使用，需相对安全环境

## 第二章：骰子系统

### 2.1 核心机制：d100 + d10

本系统采用 **目标数判定**（Target Number, TN）机制。

**基本判定流程：**
1. 玩家声明行动
2. DM确定对应属性+技能，计算 **基础值** = 属性 × 10 + 技能等级 × 5
3. DM根据情境设定 **难度修正**
4. 玩家投 **1d100**
5. 投出结果 ≤ 基础值+修正 = **成功**，否则 **失败**

### 2.2 难度等级

| 难度 | 修正值 | 描述 |
|------|--------|------|
| 轻松 | +30 | 几乎不可能失败 |
| 普通 | +10 | 稍有挑战 |
| 标准 | ±0 | 正常难度 |
| 困难 | -10 | 需要一定能力 |
| 极难 | -20 | 高手才能完成 |
| 噩梦 | -30 | 几乎不可能 |
| 奇迹 | -50 | 需要超凡运气 |

### 2.3 特殊投骰结果

| 结果 | 条件 | 效果 |
|------|------|------|
| **大成功** | 投出 01-05 | 行动以最佳方式成功，额外获得一个有利效果 |
| **大失败** | 投出 96-00（基础值>100时缩小为99-00） | 行动灾难性失败，可能产生负面后果 |
| **临界成功** | 投出值恰好等于基础值 | 勉强成功，但有小瑕疵 |

### 2.4 对抗检定

双方各投 1d100，与各自基础值比较：
- 双方都成功：比较**成功余量**（基础值 - 投出值），余量大者胜
- 一方成功一方失败：成功者胜
- 双方都失败：维持现状或比较失败程度

### 2.5 伤害骰

| 武器类型 | 伤害骰 | 示例 |
|----------|--------|------|
| 轻型武器 | 1d6 | 小刀、短棍 |
| 中型武器 | 1d8 | 撬棍、球棒、手枪 |
| 重型武器 | 1d10 | 消防斧、步枪 |
| 超重型武器 | 2d8 | 霰弹枪、双手重锤 |
| 异能攻击 | 见异能描述 | 随异能而定 |
| 丧尸爪击 | 1d6+感染 | 附带感染检定 |
| Boss攻击 | 2d10+ | 随Boss而定 |

## 第三章：技能系统

### 3.1 技能列表与关联属性

每个技能有 **0-5** 级。0级=未受训（使用纯属性值/2检定），1-5级逐步精通。

**战斗技能：**
| 技能 | 关联属性 | 描述 |
|------|----------|------|
| 近战格斗 | STR | 近战武器和徒手搏斗 |
| 射击 | AGI | 火器和投掷武器 |
| 格挡 | STR/CON | 用武器或盾牌格挡攻击 |
| 闪避 | AGI | 主动躲避攻击和危险 |

**生存技能：**
| 技能 | 关联属性 | 描述 |
|------|----------|------|
| 急救 | INT | 止血、包扎、简单医疗 |
| 搜索 | PER | 搜寻物资、发现线索 |
| 潜行 | AGI | 安静移动、避免被发现 |
| 求生 | CON/PER | 野外生存、方向判断 |
| 锁技 | AGI/INT | 撬锁、破解简单机关 |

**知识技能：**
| 技能 | 关联属性 | 描述 |
|------|----------|------|
| 机械 | INT | 修理装备、操作机械 |
| 电子 | INT | 操作电子设备、黑客 |
| 医学 | INT | 高级医疗、药物知识 |
| 博学 | INT | 历史、科学等杂学知识 |

**社交技能：**
| 技能 | 关联属性 | 描述 |
|------|----------|------|
| 说服 | WIL/INT | 逻辑说服他人 |
| 威吓 | STR/WIL | 用威慑力压制对方 |
| 察言观色 | PER/WIL | 读取他人情绪和意图 |
| 领导 | WIL | 指挥团队、鼓舞士气 |

### 3.2 技能等级与点数

- 初始技能点：**20点**
- 升1级花费1点，升2级花费2点，升3级花费3点，升4级花费5点，升5级花费8点
- 5级为大师级，全点满一个技能需要 1+2+3+5+8 = **19点**

### 3.3 技能检定公式

**基础值 = 关联属性 × 10 + 技能等级 × 5**
**攻击基础值 = 关联属性 × 10 + 技能等级 × 5 + (AGI + PER) × 2**
（攻击额外获得敏捷+感知的命中加成，反映反应速度和破绽捕捉能力。无封顶；基础值>100时大失败范围缩小为99-00）

示例：STR 5 + 近战格斗 3 级 + AGI 5 + PER 5 → 基础值 = 50 + 15 + 20 = 85，投d100 ≤ 85即成功

## 第四章：战斗系统

### 4.1 战斗轮

每轮约代表 **6秒** 现实时间。每轮流程：

1. **先攻阶段**：所有参战者投先攻值（AGI + PER + 1d10），从高到低排列
2. **行动阶段**：按先攻顺序，每人拥有 **1个主要行动 + 1个次要行动 + 自由移动**
3. **结算阶段**：处理持续效果（流血、中毒、燃烧等）

### 4.2 行动类型

**主要行动（每轮1次）：**
- 攻击（近战/远程/异能）
- 使用道具（注射药物、投掷物品）
- 全力防御（本轮闪避/格挡+20）
- 全力冲刺（移动距离翻倍，放弃攻击）
- 施展复杂异能

**次要行动（每轮1次）：**
- 换武器/装备
- 快速检查环境
- 简短对话/指挥
- 施展简单异能
- 掩护移动

**自由行动：**
- 移动 AGI × 2 米
- 倒地/站起
- 喊话

### 4.2.1 借机攻击（Opportunity Attack）

当一个角色**在敌人的近战范围内（2m）移动离开**时，该敌人可以立刻进行一次 **借机攻击**（不消耗行动次数，每轮每个敌人最多触发1次）。

**触发条件：**
- 移动者从敌人近战范围（2m内）移动到范围外
- 移动者**未使用**全力冲刺、闪避移动等脱离手段

**判定方式：**
- 攻击检定：敌人正常近战攻击基础值
- 防御：移动者**不能选择闪避或格挡**（背对/注意力不在防御上），视为不防御
- 伤害：普通近战伤害（非偷袭，不含伏击加成）

**不触发借机攻击的情况：**
- 使用"全力冲刺"作为主要行动后的移动（已放弃攻击换取安全脱离）
- 使用"掩护移动"（次要行动）后的移动
- 异能效果明确注明"不触发借机攻击"（如爆发冲刺、短距传送）
- 敌人处于眩晕、倒地、失明等无法反应的状态

> **DM注**：本规则为核心规则书 v1.1 补充条目。异能"爆发冲刺"和"短距传送"已引用此机制。

---

### 4.3 攻击判定流程

```
1. 攻击者投 1d100 vs 攻击基础值
2. 若命中 → 防御者选择：
   a. 闪避：投 1d100 vs 闪避基础值（成功则完全回避）
   b. 格挡：投 1d100 vs 格挡基础值（成功则减少一半伤害）
   c. 不防御：直接承受伤害
3. 若最终命中 → 投伤害骰
4. 伤害 = 伤害骰结果 + 力量修正（近战, STR-5）+ 技能伤害加成（近战格斗/射击等级×1）- 护甲值
5. 大成功（01-05）命中时：伤害骰投两次取大值，额外+3伤害
6. 最终伤害不低于1（只要命中就至少造成1点伤害）
```

### 4.3.1 部位瞄准（Called Shot）

角色可以声明攻击敌人的特定部位，以牺牲命中率换取额外效果。

**规则：**
- 部位瞄准消耗 **主要行动**（与普通攻击相同）
- 命中检定受 **命中惩罚**，伤害或效果获得 **额外加成**
- 可与蓄力攻击叠加（同时放弃次要行动）

| 部位 | 命中修正 | 效果 | 备注 |
|------|----------|------|------|
| **头部** | -20 | 伤害×1.5（向下取整） | 对无头盔目标；丧尸头部是已知弱点 |
| **四肢** | -10 | 命中肢体可能导致缴械（武器手）或跌倒（腿部），由DM裁决 | 不增加伤害 |
| **躯干** | ±0 | 默认攻击部位，无额外效果 | 不需要声明 |

**与弱点利用的关系：**
- 若通过博学/感知成功分析过敌人弱点（如"丧尸比较呆，头部不会躲闪"），头部瞄准的命中惩罚从-20减轻为 **-10**
- 弱点利用的攻击+10与部位瞄准的命中惩罚分别计算，可叠加

**示例：**
桑吉尔夫蓄力攻击丧尸头部（已知弱点）：基础值125 + 蓄力+5 + 头部瞄准-10（弱点已知减轻） + 弱点利用+10 = 130。伤害 = (武器骰+修正) × 1.5

> **DM注**：本规则为核心规则书 v1.2 补充条目。追溯适用于R23、R24、R35的头部攻击判定。此前使用的"头部-10命中、伤害×2"惯例正式修订为上表规则。

### 4.4 特殊战斗状态

| 状态 | 效果 | 持续 |
|------|------|------|
| **流血** | 每轮损失 1d4 HP | 直到急救成功 |
| **中毒** | 每轮损失 2 HP，所有检定-5 | CON检定成功或使用解毒剂 |
| **感染** | 每小时 CON 检定，失败则变异进展+1 | 需要特殊药物 |
| **恐惧** | 所有检定-10，不能主动靠近恐惧源 | WIL检定DC15成功 |
| **疲惫** | 所有物理检定-10，移动减半 | 休息1小时 |
| **眩晕** | 跳过下一轮行动 | 1轮 |
| **倒地** | 近战攻击对其+15，闪避-10 | 消耗次要行动站起 |

### 4.5 丧尸感染系统

被丧尸攻击命中后，需进行 **感染检定**：

1. 投 1d100 vs CON × 10
2. 失败：获得 **感染值 +1**（每次被咬/抓都要检定）
3. 感染等级表：

| 感染值 | 阶段 | 效果 |
|--------|------|------|
| 1-2 | 潜伏 | 无明显症状，每6小时CON检定防止恶化 |
| 3-4 | 初期 | CON-1，PER-1，偶尔头痛 |
| 5-6 | 中期 | CON-2，所有属性-1，开始出现幻觉 |
| 7-8 | 晚期 | 所有属性-2，每小时WIL检定防止失控 |
| 9 | 末期 | 几乎失控，需立即治疗 |
| 10 | 转化 | 角色变为丧尸，游戏结束 |

**治疗：**
- 潜伏/初期：普通抗生素（感染值-1）
- 中期：特效药（感染值-2，稀有物资）
- 晚期/末期：只有实验室的血清能治愈（目标地点）

## 第五章：异能系统

### 5.1 异能觉醒

在丧尸病毒爆发后，部分人类觉醒了异能。异能消耗 **SP（精神值）**。

**觉醒方式：**
- 角色创建时选择1个异能
- 游戏中通过特殊事件可能觉醒新异能（最多持有3个）

### 5.2 异能等级

每个异能有 **1-3级**，等级越高解锁更多用法。

升级条件：累计使用该异能 10/25/50 次，或经历特殊事件。

### 5.3 异能使用规则

- 每个异能有**多种用法**，分为**主动**（主要行动）和**快速**（次要行动）
- 持续性效果需要在角色卡中记录**剩余轮数**
- 同一异能的不同用法共享冷却：同一轮内只能使用同一异能的一种用法
- 不同异能可以在同一轮内组合（一个用主要行动，另一个用次要行动的快速用法）

### 5.4 基因锁突破系统

> 人类的基因中沉睡着远超日常表现的潜能。在极端压力、濒死危机、或灵魂级别的角色扮演时刻，DM可以对角色发起**基因锁突破判定**——一次打破生理极限的机会。

**触发条件（满足任意一条，由DM判定）：**

| 触发类型 | 描述 | 示例 |
|----------|------|------|
| 🔴 **濒死困境** | HP≤最大值20%，或感染值≥7，或陷入必死局面 | 小鱼感染5+倒地+脱手，独面行尸 |
| 🟠 **绝境反击** | 队伍面临团灭级威胁，角色挺身而出承担关键行动 | 独自断后掩护队友撤退 |
| 🟡 **超绝角色扮演** | 玩家做出极具感染力、完全贴合角色灵魂的行为 | 角色在崩溃边缘做出违背本能但符合信念的选择 |
| 🔵 **异能共鸣** | 异能使用次数≥8次且在高压场景中使用异能 | 陆第9次使用感知扩张时恰好面对未知威胁 |

**判定流程：**

```
1. DM宣告"基因锁判定"（可在叙事中自然嵌入，不必打断节奏）
2. 角色投 1d100 vs WIL×8（基础目标值）
3. 根据触发强度，DM给予修正：
   - 濒死困境：+10（绝境更容易突破）
   - 绝境反击：+5
   - 超绝角色扮演：+15（最大奖励给最好的扮演）
   - 异能共鸣：+0（自然突破，不加修正）
4. 大成功（≤5）：完美突破，获得最高级奖励
5. 普通成功：突破成功，获得标准奖励
6. 失败：未能突破，但获得 +5 EXP（挣扎本身有价值）
7. 大失败（96-100）：反噬——SP归零，昏迷1d4轮
```

**突破奖励表（DM根据情境选择最合适的一项）：**

| 奖励类型 | 效果 | 适用场景 |
|----------|------|----------|
| 🌟 **异能觉醒** | 觉醒第2或第3异能（从异能列表中选择） | 角色只有1个异能，且触发场景与新异能主题契合 |
| ⬆️ **异能升级** | 当前异能立即升1级（无视使用次数要求） | 异能使用次数≥5，且场景中正在使用该异能 |
| 💪 **属性突破** | 1个属性永久+1（突破角色创建上限10） | 物理/精神极限场景 |
| 🛡️ **特殊抗性** | 获得永久被动：某类伤害-2，或某类检定+5 | 反复承受同类威胁后的适应 |
| 🔥 **极限爆发** | 本场景内：所有检定+20，伤害+5，免疫恐惧。场景结束后HP/SP减半 | 团灭级危机中的最后一搏 |

**大成功额外效果：** 在上述奖励基础上，额外获得以下之一：
- +20 EXP
- SP完全恢复
- 感染值-1（身体在突破中短暂压制了病毒）

**限制：**
- 每个角色每个游戏日最多触发 **1次** 基因锁判定
- DM不应滥用此机制——它是**稀有的高光时刻**，不是常规升级途径
- 玩家不能主动要求触发，只有DM根据叙事判断是否满足条件
- 触发时必须有合理的叙事铺垫，不能凭空出现

> **DM注：** 基因锁突破是奖励玩家创造精彩瞬间的机制。宁可少用不可滥用。最好的触发是玩家自己都没意识到自己在做突破级别的事——然后DM给出惊喜。

### 5.5 异能副作用

**SP=0强行使用：**
1. 投 1d100，≤30：异能失控，随机负面效果
2. 31-70：异能正常发动但受 2d6 真实伤害
3. 71-00：异能失败，陷入眩晕2轮

**持续效果管理：**
所有有持续轮数的异能效果必须在角色卡的"活跃增益/减益"栏中记录，每轮结算阶段倒计时-1。归零时效果自动结束（包括附加的副作用，如疲惫）。

### 战斗速算

```
命中 = 1d100 ≤ (属性×10 + 技能×5 + 修正)
伤害 = 武器骰 + STR修正(STR-5,近战) + 技能等级×1 - 护甲（最低1）
大成功伤害 = 伤害骰投两次取大 + 上述修正 + 3
先攻 = AGI + PER + 1d10
```
