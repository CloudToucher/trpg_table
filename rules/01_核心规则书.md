## 第一章：数值体系

### 1.1 基础属性（共6项，初始点数池：30点）

每项属性范围 **1-10**（人类常规极限），超过10需要异能或特殊装备加成。

| 属性 | 缩写 | 描述 | 影响领域 |
|------|------|------|----------|
| **力量** | STR | 物理力量、近战伤害、负重 | 近战攻击、破坏、搬运 |
| **敏捷** | AGI | 速度、反应、闪避 | 闪避、先攻、远程命中 |
| **体质** | CON | 耐久、生命、抗性 | 生命值、毒/病抗性、耐力 |
| **感知** | PER | 观察、直觉、搜索 | 侦查、陷阱发现、伏击察觉 |
| **智力** | INT | 分析、记忆、技术 | 技术操作、解谜、异能控制 |
| **意志** | WIL | 精神力、抗压、异能潜力 | 精神防御、异能强度、恐惧抗性 |

### 1.2 衍生数值

| 数值 | 计算公式 | 说明 |
|------|----------|------|
| **生命值 HP** | CON × 5 + 10 | 降至0则濒死，-10则死亡 |
| **精神值 SP** | WIL × 4 + INT × 2 | 使用异能和承受恐惧消耗 |
| **耐力值 EP** | CON × 3 + STR × 2 | 奔跑、格挡、连续战斗消耗 |
| **先攻值** | AGI + PER + 1d10 | 每轮战斗开始时投骰决定行动顺序 |
| **近战攻击** | STR + 技能修正 | 基础近战命中加值 |
| **远程攻击** | AGI + 技能修正 | 基础远程命中加值 |
| **闪避值** | AGI + PER/2(向下取整) | 被动闪避门槛 |
| **负重上限** | STR × 5 (kg) | 超出则AGI-2，速度减半 |

### 1.3 伤害与治疗

**伤害类型：**
- **物理伤害**：受护甲减免
- **异能伤害**：受精神抗性减免
- **真实伤害**：无法减免（毒素、诅咒等）

**濒死与死亡：**
- HP降至0：濒死状态，每轮投1d10，≥6则稳定，＜6则HP-1
- HP降至-10：角色死亡
- 濒死状态可被治疗/急救拉回（INT检定DC12）

**自然恢复：**
- 休息1小时：恢复 CON 点HP，WIL 点SP，CON+STR 点EP
- 完整睡眠（4小时以上）：恢复 50% 最大HP/SP/EP
- 进食+饮水：额外恢复 10HP，5EP

## 第二章：骰子系统

### 2.1 核心机制：d100 + d10

本系统采用 **目标数判定**（Target Number, TN）机制。

**基本判定流程：**
1. 玩家声明行动
2. DM确定对应属性+技能，计算 **基础值** = 属性 × 10 + 技能等级 × 5
3. DM根据情境设定 **难度修正**
4. 玩家投 **1d100**
5. 投出结果 ≤ 基础值+修正 = **成功**，否则 **失败**

### 2.2 难度等级

| 难度 | 修正值 | 描述 |
|------|--------|------|
| 轻松 | +30 | 几乎不可能失败 |
| 普通 | +10 | 稍有挑战 |
| 标准 | ±0 | 正常难度 |
| 困难 | -10 | 需要一定能力 |
| 极难 | -20 | 高手才能完成 |
| 噩梦 | -30 | 几乎不可能 |
| 奇迹 | -50 | 需要超凡运气 |

### 2.3 特殊投骰结果

| 结果 | 条件 | 效果 |
|------|------|------|
| **大成功** | 投出 01-05 | 行动以最佳方式成功，额外获得一个有利效果 |
| **大失败** | 投出 96-00 | 行动灾难性失败，可能产生负面后果 |
| **临界成功** | 投出值恰好等于基础值 | 勉强成功，但有小瑕疵 |

### 2.4 对抗检定

双方各投 1d100，与各自基础值比较：
- 双方都成功：比较**成功余量**（基础值 - 投出值），余量大者胜
- 一方成功一方失败：成功者胜
- 双方都失败：维持现状或比较失败程度

### 2.5 伤害骰

| 武器类型 | 伤害骰 | 示例 |
|----------|--------|------|
| 轻型武器 | 1d6 | 小刀、短棍 |
| 中型武器 | 1d8 | 撬棍、球棒、手枪 |
| 重型武器 | 1d10 | 消防斧、步枪 |
| 超重型武器 | 2d8 | 霰弹枪、双手重锤 |
| 异能攻击 | 见异能描述 | 随异能而定 |
| 丧尸爪击 | 1d6+感染 | 附带感染检定 |
| Boss攻击 | 2d10+ | 随Boss而定 |

- 5级为大师级，全点满一个技能需要 1+2+3+5+8 = **19点**

### 3.3 技能检定公式

**基础值 = 关联属性 × 10 + 技能等级 × 5**

示例：STR 6 + 近战格斗 3 级 → 基础值 = 60 + 15 = 75，投d100 ≤ 75即成功

## 第四章：战斗系统

### 4.1 战斗轮

每轮约代表 **6秒** 现实时间。每轮流程：

1. **先攻阶段**：所有参战者投先攻值（AGI + PER + 1d10），从高到低排列
2. **行动阶段**：按先攻顺序，每人拥有 **1个主要行动 + 1个次要行动 + 自由移动**
3. **结算阶段**：处理持续效果（流血、中毒、燃烧等）

### 4.2 行动类型

**主要行动（每轮1次）：**
- 攻击（近战/远程/异能）
- 使用道具（注射药物、投掷物品）
- 全力防御（本轮闪避/格挡+20）
- 全力冲刺（移动距离翻倍，放弃攻击）
- 施展复杂异能

**次要行动（每轮1次）：**
- 换武器/装备
- 快速检查环境
- 简短对话/指挥
- 施展简单异能
- 掩护移动

**自由行动：**
- 移动 AGI × 2 米
- 倒地/站起
- 喊话

### 4.3 攻击判定流程

```
1. 攻击者投 1d100 vs 攻击基础值
2. 若命中 → 防御者选择：
   a. 闪避：投 1d100 vs 闪避基础值（成功则完全回避）
   b. 格挡：投 1d100 vs 格挡基础值（成功则减少一半伤害）
   c. 不防御：直接承受伤害
3. 若最终命中 → 投伤害骰
4. 伤害 = 伤害骰结果 + 力量修正（近战）- 护甲值
5. 最终伤害不低于1（只要命中就至少造成1点伤害）
```

### 4.4 特殊战斗状态

| 状态 | 效果 | 持续 |
|------|------|------|
| **流血** | 每轮损失 1d4 HP | 直到急救成功 |
| **中毒** | 每轮损失 2 HP，所有检定-5 | CON检定成功或使用解毒剂 |
| **感染** | 每小时 CON 检定，失败则变异进展+1 | 需要特殊药物 |
| **恐惧** | 所有检定-10，不能主动靠近恐惧源 | WIL检定DC15成功 |
| **疲惫** | 所有物理检定-10，移动减半 | 休息1小时 |
| **眩晕** | 跳过下一轮行动 | 1轮 |
| **倒地** | 近战攻击对其+15，闪避-10 | 消耗次要行动站起 |

### 4.5 丧尸感染系统

被丧尸攻击命中后，需进行 **感染检定**：

1. 投 1d100 vs CON × 10
2. 失败：获得 **感染值 +1**（每次被咬/抓都要检定）
3. 感染等级表：

| 感染值 | 阶段 | 效果 |
|--------|------|------|
| 1-2 | 潜伏 | 无明显症状，每6小时CON检定防止恶化 |
| 3-4 | 初期 | CON-1，PER-1，偶尔头痛 |
| 5-6 | 中期 | CON-2，所有属性-1，开始出现幻觉 |
| 7-8 | 晚期 | 所有属性-2，每小时WIL检定防止失控 |
| 9 | 末期 | 几乎失控，需立即治疗 |
| 10 | 转化 | 角色变为丧尸，游戏结束 |

**治疗：**
- 潜伏/初期：普通抗生素（感染值-1）
- 中期：特效药（感染值-2，稀有物资）
- 晚期/末期：只有实验室的血清能治愈（目标地点）

## 第五章：异能系统

### 5.1 异能觉醒

在丧尸病毒爆发后，部分人类觉醒了异能。异能消耗 **SP（精神值）**。

**觉醒方式：**
- 角色创建时选择1个异能
- 游戏中通过特殊事件可能觉醒新异能（最多持有3个）

### 5.2 异能等级

每个异能有 **1-3级**，等级越高效果越强、消耗越大。

升级条件：累计使用该异能 10/25/50 次，或经历特殊事件。

### 7.1 资源管理

**饥饿系统：**
- 每 8小时 不进食：获得饥饿等级+1
- 饥饿1级：EP恢复减半
- 饥饿2级：所有物理检定-5
- 饥饿3级：每小时损失1HP
- 饥饿4级：所有检定-15，每小时损失2HP

- 每 6小时 不饮水：获得口渴等级+1
- 口渴效果比饥饿更严重，每级额外-5检定

每次产生噪音，DM投 **1d100**：
- 安静区域：≤10 吸引丧尸
- 普通区域：≤30 吸引丧尸
- 危险区域：≤50 吸引丧尸
- 丧尸密集区：≤70 吸引丧尸

吸引数量：1d6（普通区域）到 3d6（密集区域）
