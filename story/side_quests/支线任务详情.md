## S01：失踪的孩子（已修订 — 隧道补救版）

### 原始触发（已错过）
- 位置：序章 - 东华路地铁站
- 条件：查看公告板 或 与刘姐交谈
- 状态：**队伍已离开东华路站，未触发**

### 补救触发
- **位置**：第一章 - SUB-002隧道（人民广场站前100m附近）
- **条件**：PER搜索成功 或 队伍经过时听到微弱哭声（DM暗骰PER DC标准）
- **时机**：队伍距人民广场站约100-200m时自动触发暗骰

### 修订描述
隧道侧方维修通道深处传来断断续续的哭声。走近后发现一个8岁左右的男孩——小宇——蜷缩在电力维修间的角落里，浑身发抖。他是从东华路站避难所跑出来的，想找爸爸，结果迷路了。他已经在这里待了几个小时，又冷又饿。

### 修订目标
在隧道中发现并救助小宇，决定如何处置他

### 修订流程
1. 听到哭声 → 队伍决定是否调查侧方维修通道
2. 进入维修通道（约30m），找到小宇
3. 小宇被卡在倒塌的工具架下 → STR检定DC标准搬开
4. 小宇极度脱水 → 需要给水（消耗瓶装水×1）
5. 安抚小宇 → 说服/领导 DC标准（成功则小宇提供信息）
6. 决定处置：
   - A：带上小宇一起前进（负重+15kg，但小宇会在人民广场站提供隐藏通道信息）
   - B：让小宇原地等待，标记位置（回程时再接）
   - C：指引小宇返回东华路站方向（WIL检定DC困难让他独自返回，有风险）
7. 回程时维修通道口出现1只行尸（保护小宇的战斗）

### 修订奖励
- 参与者各+20 EXP
- 给小宇水和食物：额外+5 EXP（善举）
- 小宇信息（安抚成功后）：
  - "我在通道里看到墙上有发光的石头" → 异能水晶线索
  - "前面那个大站台里有个叔叔一直在敲门" → **S03线索提前获得**
  - 如果带着小宇到达人民广场站：S03触发时小宇帮忙指路（搜索+10）

### 失败后果
- 不调查：无负面，但错过奖励和S03线索
- 小宇受伤/死亡：队伍SP-3（目睹孩子受伤的精神冲击）

## S02：军火库（已修订 — 植入当前路线）

### 触发
- **位置**：第一章 - SUB-002隧道（队伍当前位置~1.9km附近）
- **条件**：PER搜索成功（DC标准）发现侧壁军方标记 **或** 桑的前方警戒成功时DM主动描述
- **时机**：R11或R12，队伍继续前进时自动植入

### 修订描述
隧道右侧墙壁上出现了用反光漆喷涂的军方标记——一个箭头指向侧方的铁门，下方写着"TP-07"。铁门半掩着，门缝里传来一股火药和机油混合的气味。

### 目标
找到并打开军方物资箱

### 修订流程
1. 发现军方标记 → 队伍决定是否进入
2. 铁门需要用力推开（STR DC标准）或从缝隙挤入（AGI DC标准）
3. 侧通道约15m，尽头是一个3m×4m的小型仓库
4. 仓库情况：
   - 一具穿军装的尸体靠墙坐着（已死亡，非感染）→ WIL DC标准避免SP-2
   - 上锁的武器柜（锁技DC困难 **或** 电子DC困难破解密码锁 **或** STR DC极难暴力）
   - 弹药箱（已开封，可直接取用）
   - 军用急救箱（未上锁）
   - 军人身上有一封信（S10碎片线索）
5. **陷阱**：武器柜开锁失败（大失败）触发警报蜂鸣器 → 1d4轮后2只奔跑者从隧道方向闻声赶来
6. 如果刘流柳在场：博学检定DC标准可判断军人死因（非病毒，是枪伤——自杀）→ 额外线索

### 修订奖励
- 参与者各+25 EXP
- **武器柜（开锁成功）**：
  - 制式手枪×1（1d8伤害，10m射程，弹匣8发） **或**
  - 战术刀×1（1d8+1伤害，锋利+穿刺，耐久25/25，★★精良）
  - （d4决定：1-2=手枪，3-4=战术刀）
- **弹药箱**：手枪弹药×12
- **急救箱**：急救包×2、抗生素×1、止痛药×1
- **军人遗物**：军用手电筒（强光版，照明15m，电池8小时）、军人ID卡（可能在后续场景有用）
- **军人信件**：S10碎片#1（替代错过的公告板碎片）— 内容为军人写给家人的遗书，提到"实验室的人让我们封锁隧道，但他们自己从另一条路跑了"

### 失败后果
- 不进入：无负面，但错过大量物资和武器升级
- 触发警报：被2只奔跑者攻击（奔跑者HP12/AGI6/先攻高，比行尸危险得多）
- 如果带着小宇(S01)进入：小宇害怕尸体会哭，可能吸引注意（噪音风险）

### DM注意
- 这是队伍获得**远程武器**的第一个机会
- 战术刀是刘流柳/陆仁甲的重大武器升级（替代小刀/铁管）
- 军人遗书是重要的叙事线索和S10碎片补救

## S03：地下医生（已修订 — 即将触发）

### 触发
- **位置**：第一章 - 人民广场站（SUB-003）
- **条件**：进入站台后PER暗骰DC标准（听到微弱敲击声）
- **时机**：队伍到达人民广场站时**自动触发**暗骰（高优先级支线）

### 修订描述
人民广场站——曾经最繁忙的换乘站。站台上散落着行李和干涸的血迹。但在东侧尽头，一扇标着"员工休息室"的门后面，传来有规律的敲击声。三下，停顿，三下。有人在用SOS的节奏敲门。

### 目标
救出被困的医生「陈医生」

### 修订流程
1. 听到敲击声（PER暗骰DC标准，如果有S01小宇则小宇主动指出）
2. 接近休息室门 → 门被铁丝从外面缠死（有人故意把他锁在里面？）
3. 解开铁丝：锁技DC标准 **或** STR DC困难硬拉 **或** 用工具（胶带+小刀等）DC标准
4. 陈医生状态：严重脱水，左腿有旧伤（被咬但已自行截断感染扩散——他切掉了自己左小腿的一块肉）
5. 紧急救治：
   - 给水（消耗瓶装水×1）→ 陈医生恢复意识
   - 急救检定DC标准 → 处理伤口，陈医生可以行走
   - 如果有抗生素（S02获取）→ 额外效果：陈医生完全恢复
6. 陈医生交代情况：
   - 他是人民广场站附近诊所的医生
   - 丧尸潮爆发后他躲进休息室，但后来有人从外面用铁丝锁住了门（谁？为什么？— 伏笔）
   - 他自己被咬伤后用手术刀切除了感染组织（医学知识救了他）
7. 陈医生加入/交易：
   - **如果队伍邀请**：陈医生可以作为临时NPC跟随（HP15，医学4级，急救3级，速度减半因腿伤）
   - **如果不邀请**：陈医生留在人民广场站，但教你一个技能作为报答

### 修订奖励
- 参与者各+20 EXP
- **技能选择（每人可选一个）**：
  - 医学 0→1（免费学习）
  - 急救 +1级（免费提升）
  - 求生 +1级（陈医生传授的野外医学知识）
- **陈医生赠送**：
  - 急救包×1（他仅剩的）
  - 止痛药×2
  - 手术刀×1（1d4+2伤害，精准+暴击范围扩大，★★精良）
- **关键信息**：
  - 感染处理方法："被咬后30分钟内切除伤口周围组织可以阻止感染扩散。感染值3以上就太晚了。"
  - 人民广场站西侧有清洁间 → **给刘流柳的化学武器原料线索**
  - "我听到换乘通道那边有金属碰撞声，像是什么东西在移动" → 提示S10碎片#2位置

### 失败后果
- 不救：无负面，但错过技能提升和关键医学信息
- 陈医生死亡（开门太晚/未给水）：目睹者SP-5

### DM注意
- **这是全队获得免费技能点的唯一早期机会**，对成长至关重要
- 刘流柳的化学武器思路在此获得原料线索
- 余小鱼的感染值=1，陈医生的感染处理信息对她是救命稻草
- "谁锁了门"这个伏笔可以在后续展开（超市据点的人？另一个幸存者团伙？）

## S03b：清洁间搜刮（微型支线 — 人民广场站）

### 触发
- **位置**：人民广场站（SUB-003）西侧
- **条件**：陈医生提及 **或** 刘流柳主动寻找化学原料 **或** PER搜索成功

### 描述
人民广场站西侧走廊尽头有一间清洁间。门没锁。里面的架子上还有大半瓶工业漂白水、清洁剂、抹布、橡胶手套。对化学专业的人来说，这些东西可以变成武器。

### 流程
1. 进入清洁间 → 自动获得清洁用品
2. 如果刘流柳在场 + 博学/化学 DC标准：
   - 成功：制作**腐蚀性投掷瓶×2**（1d6酸蚀伤害+持续1d4/轮×2轮，范围投掷3m溅射）

3. 制作耗时：约10分钟游戏时间

### 奖励
- 刘流柳+10 EXP（首次化学制作）
- 腐蚀性投掷瓶×2
- 可能的刺激性烟雾弹×1
- 橡胶手套×2（防接触感染，感染检定+10）
- 工业漂白水残余（后续可再制作1瓶）

### DM注意
- 这是**刘流柳的专属成长时刻**——他的INT8和化学知识终于有了实际产出
- 橡胶手套是防感染的重要道具，尤其对余小鱼

   - 完成0-1个任务：丧尸突破，混乱战斗

### 奖励

## S07：异能觉醒 / 升级（已修订 — 增加升级路径）

### 触发A：第二异能觉醒
- **位置**：任意（随机事件表91-95"异能反应"触发）
- **条件**：角色仅有1个异能 + 随机事件触发

### 描述
你感到体内涌起一股强烈的能量波动。这种感觉似曾相识——就像你第一次觉醒异能时一样，但这次更加强烈。

### 流程
1. WIL检定DC困难
2. 成功：你的意识被拉入一个异空间，面对三个发光的符文
3. 选择一个符文（代表不同的异能类型）
4. 觉醒新异能（第2个异能槽）
5. 失败：仅感到一阵头痛，SP-5，但下次遇到此事件时检定+10

### 奖励
- 觉醒第二个异能
- 经验+15

### 触发B：异能升级（新增）
- **条件**：累计使用某异能10次 **或** 在极端压力下使用异能（HP<30%时 / SP<20%时 / 保护队友时）
