## S01：失踪的孩子

### 触发
- **位置**：序章 - 东华路地铁站
- **条件**：查看公告板 或 与避难所中的幸存者交谈
- **NPC**：一位名叫「刘姐」的母亲

### 描述
刘姐的儿子小宇（8岁）在两天前跑进了站台尽头的维修通道，至今没有回来。她请求帮忙寻找。

### 目标
在地铁站台下方的维修通道中找到小宇并带回

### 流程
1. 进入维修通道（站台层角落的暗门）
2. 通道分叉：左边是排水通道，右边是电力维修间
3. 小宇在电力维修间里，被卡在倒塌的架子下
4. 救出检定：STR检定 DC标准（搬开架子）
5. 回程时遭遇1只行尸（从排水通道爬入）

### 奖励
- 经验+20
- 刘姐好感+3
- 刘姐赠送：罐头×2（她最后的存粮）
- 小宇告诉你他在通道里看到了"发光的石头"（异能水晶线索）

### 失败后果
- 如果不接受任务：无负面效果，但错过奖励
- 如果小宇受伤/死亡：刘姐好感-5，避难所其他人好感-2

## S02：军火库

### 触发
- **位置**：第一章 - 地铁隧道（SUB-002到SUB-003之间）
- **条件**：在隧道中发现标有军方标记的侧通道

### 描述
隧道墙壁上有一个被军方标记标注的岔路。据小林提过，军方曾在隧道中设立过临时据点。

### 目标
找到并打开军方物资箱

### 流程
1. 进入标记的侧通道
2. 通道尽头有一个上锁的铁门（锁技DC困难 或 STR DC极难暴力开门）
3. 门后是一个小型军事仓库
4. 仓库中有：
   - 上锁的武器柜（锁技DC困难）
   - 弹药箱（已开封）
   - 军用急救箱
5. 但仓库中有2只奔跑者（前军人感染体，穿着军装）

### 奖励
- 经验+25
- 武器柜：步枪×1 或 霰弹枪×1（随机）
- 弹药箱：对应弹药×20
- 急救箱：急救包×2、抗生素×1
- 军用手电筒（强光版，照明15m，电池8小时）

## S03：地下医生

### 触发
- **位置**：第一章 - 人民广场站（SUB-003）
- **条件**：搜索站台东侧时听到微弱的呼救声

### 描述
在人民广场站的一个锁着的员工休息室里，一名医生被困已有两天。

### 目标
救出被困的医生「陈医生」

### 流程
1. 听到员工休息室门后的微弱敲击声
2. 门被从外面用铁丝绑住（锁技DC标准 或 STR DC困难拉开）
3. 陈医生在里面，脱水且虚弱（需要给水和食物）
4. 如果治疗他（急救检定DC标准），他恢复后可以：
   - 教授你医学技能（医学0→1，免费）
   - 或者提升你的急救技能（急救+1级，免费）

### 奖励
- 经验+20
- 免费技能提升（医学0→1 或 急救+1级，二选一）
- 陈医生赠送：急救包×1、止痛药×2
- 陈医生提供的信息：感染后期的处理方法

## S04：超市危机

### 触发
- **位置**：第二章 - 超市据点（SUR-002）
- **条件**：进入超市后与铁头交谈

### 描述
超市据点的西侧围墙出现了裂缝，铁头担心丧尸群会在夜间突破。他需要人手帮忙加固防御。

### 目标
在丧尸群到达前加固超市西墙

### 流程
1. 铁头说明情况：预计丧尸群将在4小时后到达
2. 你需要在4小时内完成以下任务（至少完成2个）：
   - 加固围墙（机械检定DC标准 + 消耗材料）
   - 在墙外设置陷阱（机械检定DC困难）
   - 在墙外设置噪音诱饵引开部分丧尸（电子检定DC困难）
   - 组织据点内的人员防御阵型（领导检定DC标准）
3. 4小时后丧尸群到达：
   - 完成0-1个任务：丧尸突破，混乱战斗
   - 完成2个任务：丧尸被阻挡，仅需处理少量突破个体
   - 完成3-4个任务：丧尸被完全挡住，大获全胜

### 奖励
- 经验+30（基础）+10/额外任务
- 铁头好感+4
- 超市交易权限（可以自由交易）
- 在超市免费休息1晚
- 铁头赠送：球棒★★（精良）或 罐头×5

## S05：广播塔

### 触发
- **位置**：第二章 - 住宅区到商业区之间
- **条件**：在高处观察时看到一座无线电塔上的指示灯仍在闪烁

### 描述
城市中的一座商业广播塔似乎还有电力。如果能到达塔顶修复设备，可能接收到实验室的广播信号。

### 目标
修复广播塔接收装置，获取实验室信号

### 流程
1. 到达广播塔底部：塔楼入口处有3只行尸
2. 攀爬至塔顶（20层楼高）：体力消耗EP-10
3. 塔顶的设备损坏：电子检定DC困难修复
4. 修复后接收到断断续续的广播：
   - 实验室仍有人存活
   - 血清储存在地下3层
   - 警告：实验室containment已失效
   - 给出一个进入实验室的备用通道信息

### 奖励
- 经验+35
- 关键情报：实验室内部信息（第四章简化）
- S10碎片+1
- 广播信息可以与其他NPC分享以获取好感

## S06：背叛者

### 触发
- **位置**：第二章 - 超市据点或学校堡垒附近
- **条件**：在夜间休息时PER暗骰成功

### 描述
夜间你注意到一个人影偷偷从据点溜出去，似乎在用某种信号装置向外发送信号。

### 目标
调查并阻止内奸

### 流程
1. 跟踪内奸（潜行检定DC困难）
2. 发现内奸正在用信号弹向一支武装幸存者团伙通风报信
3. 选择：
   - A：直接抓住内奸（战斗/说服/威吓）
   - B：追踪信号找到武装团伙的营地
   - C：向据点领导报告
4. 如果不处理：2天后武装团伙袭击据点

### 奖励
- 经验+25
- 阻止内奸：据点NPC好感+3
- 找到武装团伙营地：可以预先设伏或交涉
- 如果审讯内奸：获得周围区域的额外情报

## S07：异能觉醒

### 触发
- **位置**：任意（随机事件触发）
- **条件**：系统日志中"异能反应"事件

### 描述
你感到体内涌起一股强烈的能量波动。这种感觉似曾相识——就像你第一次觉醒异能时一样，但这次更加强烈。

### 流程
1. WIL检定DC困难
2. 成功：你的意识被拉入一个异空间，面对三个发光的符文
3. 选择一个符文（代表不同的异能类型）
4. 觉醒新异能（第2个异能槽）
5. 失败：仅感到一阵头痛，SP-5，但下次遇到此事件时检定+10

### 奖励
- 觉醒第二个异能
- 经验+15
