# 随机事件表

## 使用规则

每经过 **30分钟游戏时间** 或 **进入新区域** 时，DM投1d100查此表。
不同章节使用不同的事件表。

---

| 1d100 | 事件 | 描述 |
|-------|------|------|
| 01-30 | 无事发生 | 避难所内一切平静 |
| 31-40 | 幸存者争吵 | 两个幸存者因为物资分配争吵起来 |
| 41-50 | 外面的声音 | 路障外面传来丧尸的撞击声，值班人员紧张起来 |
| 51-60 | 新到幸存者 | 一个新的幸存者从外面逃入，带来新信息 |
| 61-70 | 物资清点 | 王姐宣布物资只够维持3天了 |
| 71-80 | 孩子哭泣 | 一个孩子因为害怕而哭泣，母亲安抚不住 |
| 81-90 | 电力波动 | 应急灯闪烁了几次，引起恐慌 |
| 91-00 | 隧道异响 | 从隧道方向传来奇怪的声音，需要有人去查看 |
